# 地灵 MBTI 人格化对话系统

## 1. 系统概述

地灵是福地的人格化 AI 精灵，通过 Spring AI 2.x 的 Function Calling 机制实现自然语言驱动的福地管理。

### 核心流程

```
玩家: "@地灵 在0,0种灵芝"
  ↓
FudiHandle → FudiCommandHandler → SpiritChatService
  ├─ 设置用户上下文 (UserContext)
  ├─ 构建系统 Prompt (含福地完整状态 + MBTI 人格 + 可用工具)
  ├─ 调用 LLM (带 tools 参数)
  │   └─ LLM 解析意图 → 决定调用 plantCrop 工具
  ├─ Spring AI 自动执行 SpiritTools.plantCrop()
  └─ LLM 根据执行结果生成人格化回复
```

---

## 2. 技术组件

| 组件                      | 职责                                     |
|-------------------------|----------------------------------------|
| `SpringAiConfig`        | ChatClient Bean 配置 (DeepSeek API)      |
| `SpiritPromptTemplates` | 3 种 Prompt 模板 (基础/详细/Function Calling) |
| `SpiritChatService`     | 对话核心，管理 Function Calling 流程            |
| `SpiritTools`           | 7 个 `@Tool` 工具函数                       |
| `UserContext`           | ThreadLocal 用户上下文传递                    |

---

## 3. 可用工具 (SpiritTools)

| 工具              | 说明                |
|-----------------|-------------------|
| `getGridStatus` | 获取网格状态（空闲坐标、已有地块） |
| `plantCrop`     | 种植作物              |
| `harvestCrop`   | 收获指定坐标            |
| `buildCell`     | 建造地块              |
| `removeCell`    | 拆除地块              |
| `sacrificeItem` | 献祭装备              |
| `feedBeast`     | 喂养灵兽              |

## 4. MBTI 人格系统

### 4.1 16 种人格分组

| 组        | 人格                     | 语气       |
|----------|------------------------|----------|
| 理性组 (NT) | INTJ, INTP, ENTJ, ENTP | 简洁、逻辑性强  |
| 理想组 (NF) | INFJ, INFP, ENFJ, ENFP | 温和、富有同理心 |
| 行动组 (SP) | ISTP, ISFP, ESTP, ESFP | 热情、直接    |
| 关怀组 (SJ) | ISTJ, ISFJ, ESTJ, ESFJ | 体贴、细致    |

### 4.2 人格示例

**INTJ**: "📊 福地状态。灵气：500/1000，效率在预期范围内。"
**ESFP**: "🔥 福地状态超棒！有什么要我帮忙的吗？✨"
**INFJ**: "🌿 福地正在运转呢～我能感受到灵气的流动。"

## 5. 配置

```yaml
spring:
  ai:
    openai:
      api-key: ${OPENAI_API_KEY:sk-placeholder}
      base-url: ${OPENAI_BASE_URL:https://api.deepseek.com}
      chat:
        options:
          model: ${OPENAI_MODEL:deepseek-chat}
          temperature: 0.7
          max-tokens: 1000

xiantao:
  spirit:
    max-tokens: 800
    enable-fallback: true
```
