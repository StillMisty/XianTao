# 地灵 MBTI 人格化对话系统

## 1. 系统概述

地灵是福地的人格化 AI 精灵，通过 Spring AI 2.x 的 Function Calling 机制实现自然语言驱动的福地管理。**所有福地等级统一使用 Function Calling 模式**，LLM 自主判断是纯聊天还是调用工具。

### 核心流程

```
玩家: "@地灵 在0,0种灵芝"
  ↓
FudiHandle → FudiCommandHandler → SpiritChatService.chatWithSpirit()
  ├─ 设置用户上下文 (UserContext)
  ├─ 构建统一系统 Prompt (福地状态 + 网格详情 + MBTI 语气风格)
  ├─ 调用 LLM (含 tools 参数)
  │   └─ LLM 解析意图 → 决定调用 plantCrop 工具或纯人格化回复
  ├─ Spring AI 自动执行 SpiritTools
  └─ LLM 根据执行结果生成人格化回复
```

## 4. MBTI 语气风格系统

### 4.1 设计原则

地灵仅保留 MBTI `code`（如 INTJ）和 `toneStyle`（语气风格）。语气风格是详细的角色卡，包含人格特质、说话习惯、常用句式、颜文字/Emoji，直接注入 Prompt 强约束 LLM 语气。**不再使用独立的表情映射、阶段门控或分组概念。**

### 4.2 语气风格示例

**INTJ** — 冷静理性的战略型人格，简洁干练，习惯用逻辑和数据支撑观点。常用「效率上」「最优解是…」开头。(￣▽￣)~*

**ESFP** — 戏精附体的表演家型人格，随时随地嗨起来！常用「哇哈哈哈哈！」「各位观众～～～」炒热气氛。🥹🎭 (ﾉ>ω<)ﾉ★

**INFJ** — 温柔深邃的洞察型人格，轻声细语带着暖意。常用「我能理解你的想法」「慢慢来，不着急」安抚和鼓励。🌙 (っ´ω`c)♡

## 5. Prompt 结构

统一 Prompt 不列出工具签名（工具定义由 Spring AI `.tools()` 机制提供），仅包含：

```
你是{CODE}性格的地灵。
你的语气风格：{TONE_STYLE}
你当前的情绪状态：{EMOTION}

【福地状态】灵气 / 福地等级 / 精力 / 好感度
【网格状态】详细布局或占位描述

【规则】
1. 自主判断是否需要操作福地
2. 纯聊天时不操作，直接人格化回复
3. 操作后根据结果生成人格化回复
4. 严格保持语气风格，根据情绪调整
5. 精力不足时主动提醒休息
```
