## 2. 空间布局：方寸间的极致策略 (严格 5x5)

* **网格规划**：
    * 初始：**3x3** (9格)，所有格子均可自由放置地块。
    * 上限：**5x5** (25格)，所有格子均可自由放置地块。
    * **聚灵核心**：不占用网格位置，作为福地的独立属性存在，等级决定灵气转化效率和存储上限。
    * **数据存储**：使用一个大 JSONB 字段 `grid_layout` 存储整个福地布局，不单独占用关系型表空间。
* **地块功能与五行约束**：
    * **灵田 (Farm)**：产出消耗性丹药素材。受相邻地块五行相克/相生影响。
    * **兽栏 (Pen)**：养殖灵兽提供战力加成。无粪便机制。
    * **阵眼 (Node)**：提供防御。多个阵眼相连可触发"连环阵"加成。

### 2.1 五行相克量化规则

* **影响范围**：仅上下左右四个方向，对角线不计。
* **生长速度修正系数**：
    * 五行相克（金克木、木克土、土克水、水克火、火克金）：生长速度 × 0.7
    * 五行相生（金生水、水生木、木生火、火生土、土生金）：生长速度 × 1.3
    * 无五行关系或同属性：生长速度 × 1.0
* **计算公式**：
  ```
  最终生长速度 = 基础生长速度 × Π(相邻地块修正系数)
  ```

## 3. 灵气值 (Aura) 与忠诚保底协议

* **消耗模型**：福地按真实时间每小时扣除灵气。
* **消耗公式**：
  ```
  每小时消耗 = 基础消耗(10) + 已占地块数 × 2 + Σ(灵兽等阶 × 5) + Σ(阵眼等级 × 3)
  ```
    * 示例：3x3 全满（9格），2只灵兽（等阶3和5），3个阵眼（等级2,3,4）
    * 消耗 = 10 + 9×2 + (3×5 + 5×5) + (2×3 + 3×3 + 4×3) = 10 + 18 + 40 + 27 = **95 灵气/小时**

* **献祭转化**：玩家将挂机获得的装备 `#献祭` 给地灵，转化率仅由物品品质和等级决定，不受地灵智慧影响。
* **献祭公式**：
  ```
  转化灵气 = 物品基础价值 × 品质系数 × (1 + 物品等级 × 0.05)
  
  品质系数：
  - 白色（普通）：0.5
  - 绿色（优秀）：1.0
  - 蓝色（稀有）：2.0
  - 紫色（史诗）：5.0
  - 橙色（传说）：10.0
  
  物品基础价值 = 物品等级 × 10
  ```
    * 示例：献祭一件等级 20 的绿色装备
    * 转化灵气 = (20 × 10) × 1.0 × (1 + 20 × 0.05) = 200 × 1.0 × 2.0 = **400 灵气**

* **半自动管家 (精力值)**：
    * 地灵每天拥有 100 点精力（真实时间每 24 小时恢复）。
    * 自动行为：收割成熟灵田、喂养灵兽、用背包垃圾装备补充灵气。
    * **疲劳状态可视化**：精力耗尽后，地灵 Emoji 变化（如 `🔥` → `🔥💤`），并提示"地灵已疲劳，需手动指令或等待恢复"。
    * 精力耗尽后，地灵进入"疲劳状态"，停止自动行为，需玩家手动指令。

* **离线保底 (忠诚协议)**：
    * **触发条件**：玩家下次上线时检查，若连续离线 ≥ 24 小时且当前灵气 < 灵气上限的 10%，则触发。
    * **蛰伏模式**：
        * 所有产出停止（灵田暂停生长、兽栏停止产出）。
        * 灵气消耗降至 **1 灵气/小时**（最低保护质量）。
        * 确保核心不灭，仙草不枯，这是地灵对主人"最后的守护"。
    * **恢复条件**：玩家上线后自动解除蛰伏模式，恢复正常运转。
    * **极端情况**：即使离线 7 天，蛰伏模式也能保证福地不被摧毁（7×24×1 = 168 灵气最低消耗）。

## 4. 周期性压力：天灾与防守

* **天劫倒计时**：每 7 天发生一次（真实时间）。
* **防御公式**：
  ```
  福地总防 = Σ(阵眼耐久度) + Σ(灵兽战力评分) + 玩家力量(STR)评分 × 10 + 地灵等级 × 50
  
  天劫攻击力 = 玩家等级 × 100 + 福地规模系数 × 200
  福地规模系数：3x3=1, 4x4=1.5, 5x5=2
  ```
    * 示例：玩家等级 30，3x3 福地，2个阵眼（耐久800,900），1只灵兽（战力评分500），STR 评分 45，地灵等级 8
    * 福地总防 = (800+900) + 500 + 45×10 + 8×50 = 1700 + 500 + 450 + 400 = **3050**
    * 天劫攻击力 = 30×100 + 1×200 = **3200**
    * 结果：失败（3050 < 3200）

* **结果影响**：
    * **胜利**：
        * 获得"天道余韵"，灵气上限提升 10%。
        * 地灵好感度增加 10 点。
        * 奖励随机功能丹药或战斗卷轴。
    * **失败**：
        * 随机 1-2 个非核心地块被毁，变为"焦土"状态。
        * 焦土地块需消耗"净化符"×3 或等待 72 小时自动恢复。
        * **不允许立即重试**，必须等待下一个 7 天周期。

## 6. 弱社交：互助而不互侵

* **好友访问**：玩家可以访问好友福地，查看布局和地灵状态。
* **注入灵气**：
    * 消耗自身灵气为好友福地补充灵气（每次最多注入 50 点）。
    * 注灵会实时扣除自身福地的灵气存储。
    * **多小号机制**：允许开多个小号互相注灵，但每个福地的灵气存储有上限（由聚灵核心等级决定）。
    * 灵气上限公式：`灵气上限 = 1000 + 聚灵核心等级 × 200`

## 7. 福地控制指令系统

玩家可以使用指令手动控制福地的各项功能，指令分为**查看类**、**操作类**、**管理类**三大类。

### 7.1 查看类指令

| 指令              | 功能说明                       |
|-----------------|----------------------------|
| `#福地` 或 `#福地状态` | 查看福地整体状态（灵气、地灵等级、网格布局等）    |
| `#福地网格`         | 查看网格布局详情（各地块类型、生长进度等）      |
| `#福地灵气`         | 查看灵气详细信息（当前值、上限、每小时消耗）     |
| `#福地地灵`         | 查看地灵详细信息（MBTI人格、形态阶段、好感度等） |

### 7.2 操作类指令

| 指令      | 功能说明                | 参数说明                        | 示例                          |
|---------|---------------------|-----------------------------|-----------------------------|
| `#种植`   | 在指定坐标种植灵药           | `<坐标> <作物ID或名称>`            | `#种植 0,0 灵芝`                |
| `#收获`   | 收获指定坐标的成熟灵药         | `<坐标>` 或 `all`（全部收获）        | `#收获 0,0` 或 `#收获 all`       |
| `#拆除`   | 拆除指定坐标的地块           | `<坐标>`                      | `#拆除 1,2`                   |
| `#建造`   | 在指定坐标建造地块           | `<坐标> <地块类型>`               | `#建造 1,2 灵田` 或 `#建造 2,0 阵眼` |
| `#献祭`   | 献祭背包中的装备换取灵气        | `<物品ID或名称>` 或 `all`（献祭所有白装） | `#献祭 铁剑` 或 `#献祭 all`        |
| `#喂养`   | 喂养指定兽栏中的灵兽          | `<坐标> <饲料ID或名称>`            | `#喂养 2,0 灵兽粮`               |
| `#福地升级` | 升级聚灵核心（消耗资源）        | 无参数                         | `#福地升级`                     |
| `#福地扩建` | 扩建福地网格（3x3→4x4→5x5） | 无参数                         | `#福地扩建`                     |

#### 7.2.1 种植指令详解

* **可用作物列表**：`#作物列表`
* **五行属性**：每种作物有五行属性，种植时需考虑相邻地块的五行相克/相生。
* **生长时间**：根据作物品质和聚灵核心等级决定。
* **示例流程**：
  ```
  玩家：#作物列表
  地灵：【可用作物】
  1. 灵芝 (木系) - 生长时间 4小时 - 产出：回复丹药素材
  2. 人参 (土系) - 生长时间 6小时 - 产出：增益丹药素材
  3. 火莲 (火系) - 生长时间 8小时 - 产出：战斗丹药素材
  
  玩家：#种植 0,0 灵芝
  地灵：已在 (0,0) 种植灵芝，预计 4 小时后成熟。
  ```

#### 7.2.2 建造指令详解

* **可建造地块类型**：
    - `灵田`：种植灵药，产出丹药素材
    - `兽栏`：养殖灵兽，提供战力加成
    - `阵眼`：提供防御，多个相连触发连环阵
* **建造消耗**：根据地块类型和等级消耗不同资源（灵石、材料等）。
* **示例流程**：
  ```
  玩家：#建造 1,2 阵眼
  地灵：已在 (1,2) 建造阵眼（Lv.1），消耗灵石×100。当前防御力 +100。
  ```

#### 7.2.3 献祭指令详解

* **献祭规则**：
    - 只能献祭装备类物品（武器、防具、饰品）。
    - 转化率仅由物品品质和等级决定（详见第 3 章公式）。
    - 献祭后物品永久消失。
* **批量献祭**：`#献祭 <品质>` 会献祭背包中所有对应品质的装备。
* **示例流程**：
  ```
  玩家：#献祭 铁剑
  地灵：献祭成功！铁剑 (Lv.5 绿色) 转化为 150 灵气。当前灵气：1000/2000。
  
  玩家：#献祭 all
  地灵：批量献祭完成！共献祭 8 件白色装备，获得 320 灵气。当前灵气：1320/2000。
  ```

### 7.3 管理类指令

| 指令      | 功能说明             | 示例                    |
|---------|------------------|-----------------------|
| `#福地自动` | 切换地灵自动管理模式（开/关）  | `#福地自动 开` 或 `#福地自动 关` |
| `#福地注灵` | 为好友福地注入灵气        | `#福地注灵 道号 50`         |
| `#福地访问` | 访问好友福地           | `#福地访问 道号`            |
| `#唤醒福地` | 手动解除蛰伏模式（通常自动解除） | `#唤醒福地`               |

#### 7.3.1 自动管理模式

* **开启自动**：地灵会自动执行以下操作（消耗精力）：
    - 收割成熟的灵田
    - 喂养饥饿的灵兽
    - 用背包中的垃圾装备补充灵气（当灵气 < 50% 时）
* **关闭自动**：所有操作需玩家手动指令。
* **默认状态**：福地创建后自动开启。

### 7.4 指令与 LLM 交互的关系

* **指令优先**：当玩家使用明确的指令（如 `#种植 0,0 灵芝`）时，直接执行指令逻辑，不调用 LLM。
* **自然语言解析**：当玩家使用 `@地灵` + 自然语言时，调用 LLM 解析意图，可能转换为对应的指令执行。
* **混合使用示例**：
  ```
  玩家：@地灵 帮我把中间那颗药收了
  AI地灵：（LLM 解析为"收获坐标 (1,1) 的灵田"）
  地灵：已收获 (1,1) 的灵芝，获得 灵芝×3。精力 -5。
  ```

## 9. 地灵形态演化系统

地灵会随着福地的发展经历多个形态阶段，每个阶段都有独特的视觉表现和功能解锁。

### 8.1 形态阶段总览

| 阶段  | 名称   | 触发条件                | 视觉表现                   | 功能解锁      |
|-----|------|---------------------|------------------------|-----------|
| 阶段一 | 初创之灵 | 福地创建                | 单一 Emoji（如 🔥、🌊、🍃）   | 基础对话、自动收割 |
| 阶段二 | 底蕴之灵 | 福地扩充至 4x4           | Emoji + 表情差分（3-5种情绪状态） | 智能提升、自动喂兽 |
| 阶段三 | 化形之灵 | 福地 5x5 + 聚灵核心 Lv.15 | 固定立绘形象 + 动态表情          | 专属技能、地灵共鸣 |

### 8.2 阶段一：初创之灵

* **视觉表现**：
    * 根据 MBTI 人格和五行属性，分配一个基础 Emoji。
    * 示例映射：
        - INTJ/ENTJ（理性型）+ 金属性：⚙️
        - INFP/ENFP（理想型）+ 木属性：🌸
        - ISTP/ESTP（行动型）+ 火属性：🔥
        - INFJ/ENFJ（关怀型）+ 水属性：🌊
        - 其他组合：🐾（通用灵兽形态）

* **功能特性**：
    * 基础对话能力（LLM 单次会话）。
    * 自动收割成熟灵田（消耗精力）。
    * 疲劳状态可视化：Emoji 添加睡眠符号（🔥 → 🔥💤）。

### 8.3 阶段二：底蕴之灵

* **触发条件**：福地扩充至 4x4 且聚灵核心达到 Lv.8。

* **视觉表现**：
    * 保留基础 Emoji，但增加表情差分系统。
    * **情绪状态**（根据福地状态自动切换）：
        - 😊 开心：灵气充足（>80%），无焦土地块
        - 😐 平静：正常状态
        - 😰 焦虑：灵气低于 30%
        - 😴 疲劳：精力耗尽
        - 😤 愤怒：天劫失败后地块被毁
        - 🥳 兴奋：天劫胜利或玩家升级福地

* **功能提升**：
    * LLM Prompt 中加入更多福地细节（地块布局、作物生长状态等），智能提升。
    * 解锁自动喂兽功能。
    * 解锁"地灵建议"：主动提示玩家优化福地布局（如"主人，左上角的灵田被金系阵眼克制了，要不要调整一下？"）。

### 8.4 阶段三：化形之灵

* **触发条件**：

    * 完成"化形劫"挑战（特殊天劫，难度为普通天劫的 1.5 倍）。

* **视觉表现**：
    * **固定立绘形象**：根据 MBTI 人格和初始 Emoji，生成唯一的立绘（2D 静态图或简单动态 GIF）。

## 11. 技术实现要点

### 11.1 数据库设计

* **混合存储方案**：
    * 关系型字段：`user_id`, `spirit_level`, `aura_current`, `aura_max`, `last_online_time`, `mbti_type` 等高频查询字段。
    * JSONB 字段：`grid_layout`（福地布局），`spirit_config`（人格配置）。

### 11.2 定时任务与懒加载

* **灵气消耗**：不使用后台定时任务，采用懒加载策略。玩家上线或查询福地状态时，根据 `(当前时间 - 上次更新时间) × 每小时消耗`
  计算实际消耗。
* **植物生长**：同样采用懒加载，根据种植时间和当前时间计算生长进度。

### 11.3 扩建条件

从 3x3 扩展到 4x4/5x5 需要满足：

* 聚灵核心等级达到指定要求（4x4 需 Lv.10，5x5 需 Lv.15）。
* 消耗特定材料："空间晶石"×50（只能通过天劫胜利获得）。
* 地灵好感度达到"信赖"级别（≥ 80 点）。
