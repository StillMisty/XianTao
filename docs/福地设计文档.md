# 福地系统设计文档 (内部网格制 + AI经营)

**核心设计理念**：福地是每个玩家的“私人修仙农场/塔防基地”。**获取极易，但维持和扩张宛如无底洞。** 引入轻度 XY
坐标（九宫格起步）来增加策略性。

### 1. 福地的获取 (早早入坑)

* **获取方式**：玩家达到Lv5，或完成新手引导后，村长赠送一块【破败的须弥石】。
* **初始化**：滴血认主后，获得一个 **1级福地**，并随机唤醒一个初级**地灵AI**（如：蠢萌的泥巴怪、傲娇的火精灵）。

### 2. 福地内部空间结构 (微观 XY 坐标系)

福地内部是一块网格化的土地。初始为 **3x3 (共9格)**，坐标范围从 `(1,1)` 到 `(3,3)`。

每个地块可以规划为不同的**建筑类型**：

| 地块类型           | 作用                     | 养成深度 (为什么费事)                            |
|:---------------|:-----------------------|:----------------------------------------|
| **灵田 (Farm)**  | 种植种子，产出草药、属性仙果。        | 种子极难掉落；生长期按天计算；不同植物有“相生相克”属性，种错相邻格子会减产。 |
| **兽栏 (Pen)**   | 孵化灵兽，产出特殊材料或助战。        | 每天需要消耗大量食物；清理粪便（需地灵操作）；灵兽有概率生病。         |
| **阵眼 (Node)**  | 放置阵法基石，提供福地总防御力。       | 阵法基石需要天量矿石 `#锻造`；每次抵御天劫都会掉耐久，需持续补充。     |
| **聚灵阵 (Core)** | (默认固定在中心点 2,2) 维持福地运转。 | **灵气值（Aura）**消耗的源头。                     |

### 3. 福地的核心硬核限制：【灵气值 (Aura)】

这是限制玩家无限白嫖的核心机制（类似真实世界的物业费）。

* **消耗**：福地每天运转（植物生长、灵兽存活）会按小时扣除“灵气值”。
* **枯竭惩罚**：如果灵气值为 0，植物停止生长甚至枯萎，灵兽属性下降，阵法失效。
* **补充**：玩家必须在野外疯狂挂机打怪，捡垃圾，然后将不需要的装备/材料 **“献祭”** 给福地中心转化为灵气，或者直接消耗极品“灵石”。
* *(设计目的：逼迫玩家持续出门挂机，形成“打怪->养家”的闭环。)*

### 4. 周期性生存压力：【福地灾劫】

* **机制**：每现实时间 7 天，福地会迎来一次“虚空灾劫”。
* **防守计算**：如果 `阵法总防御力 + 灵兽战力 + 玩家面板 > 灾劫强度`，则防守成功，福地灵气上限永久提升。
* **失败惩罚**：福地随机一个坐标的地块被毁（植物死亡/阵法碎裂），需要重新开垦。
* *(设计目的：长线验证玩家一周的挂机养成成果，提供紧张感。)*

### 5. 地灵 AI 交互系统 (自然语言与坐标的结合)

地灵是代替玩家执行繁琐操作的管家。对于有坐标的福地，LLM 意图识别更加重要。

* **常规指令操作 (Fallback)**：
    * `#开垦 1 1 灵田`
    * `#种植 1 1 赤火莲子`
* **@地灵 交互操作 (高级)**：
    * 玩家可以直接在群里发：“`@地灵 把那颗赤火莲子种在左上角，然后用精铁矿把防守大阵修补一下。`”
    * **LLM后台解析**：AI 解析出 “左上角” = `(1,1)`，行为=`plant`，目标=`赤火莲子`。并解析出行为=`repair`，材料=`精铁矿`。
    * **地灵回复展示**：
      🧚 **傲娇火灵**：“左上角是吧？知道了知道了，真会使唤人！阵法也给你用精铁糊上了，丑死了！”
      🌱 [系统]: 坐标(1,1) 已种植赤火莲子。
      🛡️ [系统]: 阵法耐久度恢复 300 点，消耗精铁矿 x10。

### 6. 福地的终极追求 (长线目标)

1. **扩建土地**：从 3x3 扩建到 4x4 需要海量的珍稀材料（如：虚空晶石），这可能需要玩家挂机一个月才能攒够。
2. **地灵升级/化形**：地灵吸收足够灵气后，可以从初级形态（如泥巴怪）升级为高级形态（如精灵少女），甚至可以导入**酒馆卡片系统**
   ，获得独立好感度和立绘，作为玩家的终极养成伴侣。
