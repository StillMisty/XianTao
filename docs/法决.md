# 法决系统

## 1. 法决定义 (xt_skill)

### 1.1 表结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGSERIAL | PK |
| `name` | VARCHAR(128) | 法决名称（UNIQUE） |
| `description` | TEXT | 描述 |
| `skill_type` | VARCHAR(32) | 技能类型 |
| `effect_type` | VARCHAR(32) | 效果类型 |
| `binding_type` | VARCHAR(32) | 法器绑定类型 |
| `binding_value` | VARCHAR(64) | 绑定值（法器类型或大类名） |
| `cooldown_seconds` | INT | 独立 CD（秒） |
| `damage_formula` | VARCHAR(128) | 伤害公式，如 `"wis*3 + level*2 + 20"` |
| `effect_params` | JSONB | 效果参数 |
| `level_requirement` | INT | 等级要求 |
| `require_wis` | INT | 智慧要求（可空） |
| `require_skill_id` | BIGINT | 前置法决（可空，法决树预留） |
| `tags` | JSONB | 标签（用于检索、分类） |
| `create_time` / `update_time` | TIMESTAMP | 时间戳 |

### 1.2 获取方式

| 方式 | 说明 |
|------|------|
| 法决玉简 | 从背包使用法决玉简消耗习得 |
| 任务奖励 | 悬赏等任务直接奖励法决 |
| NPC 购买 | 坊市花灵石购买法决玉简 |

## 2. 效果类型 (effect_type)

### 2.1 伤害类

| 效果 | 说明 | effect_params 示例 |
|------|------|-------------------|
| `DAMAGE` | 直接伤害 | `{"base_damage": 20}` |
| `MULTI_HIT` | 连击 | `{"hits": 3}` |
| `EXECUTE` | 斩杀（低血量额外伤害） | `{"bonus_dmg_pct": 0.5, "threshold_pct": 0.3}` |
| `LIFESTEAL` | 吸血 | `{"steal_pct": 0.15}` |

### 2.2 控制类

| 效果 | 说明 | effect_params 示例 |
|------|------|-------------------|
| `ARMOR_BREAK` | 破甲（降低防御） | `{"break_pct": 0.3, "duration_sec": 10}` |
| `SLOW` | 减速（降低攻速） | `{"slow_pct": 0.4, "duration_sec": 10}` |
| `DOT` | 持续伤害 | `{"damage_per_tick": 5, "duration_sec": 12, "tick_interval_sec": 3}` |
| `STUN` | 眩晕（跳过行动） | `{"duration_sec": 3}` |
| `FREEZE` | 冰冻（跳过行动+受伤增加） | `{"duration_sec": 3, "damage_increase_pct": 0.2}` |
| `SILENCE` | 沉默（禁止技能） | `{"duration_sec": 5}` |

### 2.3 辅助类

| 效果 | 说明 | effect_params 示例 |
|------|------|-------------------|
| `HEAL` | 治疗 | `{"heal_amount": 50}` |
| `ATTACK_BUFF` | 攻击增益 | `{"buff_pct": 0.2, "duration_sec": 15}` |
| `DEFENSE_BUFF` | 防御增益 | `{"buff_pct": 0.3, "duration_sec": 15}` |
| `SPEED_BUFF` | 速度增益 | `{"buff_pct": 0.15, "duration_sec": 15}` |

## 3. 绑定类型 (binding_type)

| 类型 | 说明 | 强度 |
|------|------|------|
| `NONE` | 不绑定法器，任何法器都能触发 | 较弱 |
| `WEAPON_TYPE` | 精确绑定（如必须装备"剑"） | 强 |
| `WEAPON_CATEGORY` | 大类绑定（如"刀兵"覆盖刀/剑/斧） | 中等 |
| `ELEMENT` | 元素绑定（预留） | — |

### 3.1 法器大类

| 大类 | 包含法器子类型 |
|------|-------------|
| 刀兵 | 刀、剑、斧 |
| 长兵 | 枪、棍 |
| 远兵 | 弓 |
| 奇兵 | 鞭、戟、锤、扇、拂尘 |

**设计原则：** 绑定法器的法决效果强于不绑定的，由此产生装备策略性——玩家需要根据地图怪物类型选择法器和配套法决。

## 4. 战斗槽位

| 等级 | 可用槽位数 |
|------|----------|
| 初始 | 1 |
| 20 级 | 2 |
| 40 级 | 3 |
| 60 级 | 4 |
| 80 级 | 5 |

法决可以无限学习，但战斗中只有已放入槽位的法决生效。槽位中的法决按 CD 自动轮播。

## 5. 已学法决 (xt_player_skill)

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGSERIAL | PK |
| `user_id` | BIGINT | FK → xt_user(id) |
| `skill_id` | BIGINT | FK → xt_skill(id) |
| `equipped` | BOOLEAN | 是否放入战斗槽位 |
| `source` | VARCHAR(32) | 习得来源：JADE(玉简) / REWARD(奖励) / INITIAL(初始) |
| `learned_at` | TIMESTAMP | 习得时间 |

UNIQUE(user_id, skill_id)：防重复学习。

## 6. 战斗自动施放

### 6.1 核心逻辑

```
在每场自动战斗（懒结算的一帧）中：
  for each 已放入槽位的法决：
    if skill.CD 已转好 and 法器绑定条件满足：
      触发法决效果
      重置 CD
```

### 6.2 设计原则

- **零消耗**：法决不消耗灵力/HP，纯 CD 限制
- **独立 CD**：每个法决各自独立冷却，互不干扰
- **自动轮播**：无优先级、无触发条件判断（除斩杀类），CD 好就用
- **法器约束**：绑定了法器类型的法决，若当前装备不匹配则跳过

## 7. 法决玉简 (SKILL_JADE)

法决玉简是一种**可堆叠物品**，通过 `ItemType.SKILL_JADE` 标识。

`ItemTemplate.properties` 中记录关联的法决 ID：

```json
{"skill_id": 1}
```

### 使用流程

玩家在背包中使用法决玉简 → 消耗物品 → 向 `xt_player_skill` 插入记录 → 习得对应法决。

### 获取方式

- 怪物掉落（在怪物模板的 `drop_table.items` 中配置为普通物品掉落）
- 任务奖励
- 坊市购买

## 8. 批量设计思路

### 8.1 同效果类型 × 不同绑定

```
DAMAGE + 剑绑定 → 御剑术（CD 30s，wis×3）
DAMAGE + 刀绑定 → 破空斩（CD 40s，wis×4）
DAMAGE + 弓绑定 → 穿云箭（CD 35s，wis×2.5）
DAMAGE + 无绑定 → 烈焰诀（CD 45s，wis×2）← 无绑定但较弱
```

### 8.2 同绑定 × 不同效果

```
剑绑定 + DAMAGE → 御剑术
剑绑定 + MULTI_HIT → 万剑诀
剑绑定 + ARMOR_BREAK → 破剑式
```

### 8.3 同效果 × 不同参数

```
DAMAGE + CD短 → 小技能（快速循环）
DAMAGE + CD长 → 大技能（高爆发）
```

### 8.4 法决数量估算

```
14 种效果 × 5 种绑定 × 3 种参数档位 = 210 种基础法决
再加上等级梯度（低/中/高）× 3 = 630 种
```

达到几百上千种并不困难。
