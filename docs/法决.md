## 2. 法决定义 (xt_skill)

### 2.1 表结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGSERIAL | PK |
| `name` | VARCHAR(128) | 法决名称（UNIQUE） |
| `effect_type` | VARCHAR(32) | 效果类型 |
| `binding_type` | VARCHAR(32) | 法器绑定类型 |
| `bind_weapon_type` | VARCHAR(32) | 精确绑定的法器子类型（可空） |
| `bind_category` | VARCHAR(16) | 大类绑定：刀兵/长兵/远兵/奇兵（可空） |
| `bind_element` | VARCHAR(16) | 元素绑定（可空，预留） |
| `cooldown_seconds` | INT | 独立 CD（秒） |
| `damage_formula` | VARCHAR(128) | 伤害公式，如 `"wis*3 + level*2 + 20"` |
| `effect_params` | JSONB | 效果参数 |
| `require_level` | INT | 等级要求 |
| `require_wis` | INT | 智慧要求（可空） |
| `require_skill_id` | BIGINT | 前置法决（可空，法决树预留） |
| `description` | TEXT | 描述 |
| `tags` | JSONB | 标签 |
| `create_time` / `update_time` | TIMESTAMP | 时间戳 |

### 2.2 获取方式

| 方式 | 说明 |
|------|------|
| 法决玉简 | 从 [法决玉简](#8-法决玉简-skill_jade) 消耗习得 |
| 任务奖励 | 悬赏等任务直接奖励法决 |
| NPC 购买 | 坊市花灵石购买法决玉简 |

## 3. 效果类型 (effect_type)

法决**只有攻击类效果**，无恢复/增益类。

| 效果 | 说明 | 懒结算表现 |
|------|------|----------|
| DAMAGE | 直接伤害 | `wis × scale + base + level` |
| ARMOR_BREAK | 破甲 | 怪物防御减 X%，持续 Y 秒 |
| SLOW | 减速 | 怪物攻速减 X%，持续 Y 秒，玩家受伤减少 |
| EXECUTE | 斩杀 | 怪物 HP < 30% 时额外造成 X% 伤害 |
| LIFESTEAL | 吸血 | 造成伤害的 X% 回复玩家 HP |
| MULTI_HIT | 连击 | 一次施法打 3 下，每下独立计算 |
| DOT | 持续伤害 | 施加 dot，每 tick 额外伤害，持续 Y 秒 |

### 3.1 effect_params JSONB

按效果类型配置参数，示例：

```json
// DAMAGE
{"base_damage": 20}

// ARMOR_BREAK
{"break_pct": 0.3, "duration_sec": 10}

// SLOW
{"slow_pct": 0.4, "duration_sec": 10}

// EXECUTE
{"bonus_dmg_pct": 0.5, "threshold_pct": 0.3}

// LIFESTEAL
{"steal_pct": 0.15}

// MULTI_HIT
{"hits": 3}

// DOT
{"damage_per_tick": 5, "duration_sec": 12, "tick_interval_sec": 3}
```

## 4. 法器绑定

### 4.1 绑定类型 (binding_type)

| 类型 | 说明 | 强度 |
|------|------|------|
| NONE | 不绑定法器，任何法器都能触发 | 较弱 |
| WEAPON_TYPE | 精确绑定（如必须装备"剑"） | 强 |
| WEAPON_CATEGORY | 大类绑定（如"刀兵"覆盖刀/剑/斧） | 强 |
| ELEMENT | 元素绑定（预留） | — |
| WEAPON_AND_ELEMENT | 双条件绑定（预留） | 极强 |

### 4.2 法器大类 (category)

| 大类 | 包含法器子类型 |
|------|-------------|
| 刀兵 | 刀、剑、斧 |
| 长兵 | 枪、棍 |
| 远兵 | 弓 |
| 奇兵 | 鞭、戟、锤、扇、拂尘 |

该字段存储在 `EquipmentTemplate.category`，法决通过 `bind_category` 匹配。

**规则**：绑定法器的法决效果强于不绑定的，由此产生装备策略性——玩家需要根据地图怪物类型选择法器和配套法决。

## 5. 战斗槽位

| 等级 | 可用槽位数 |
|------|----------|
| 初始 | 1 |
| 20 级 | 2 |
| 40 级 | 3 |
| 60 级 | 4 |
| 80 级 | 5 |

法决可以无限学习，但战斗中只有已放入槽位的法决生效。槽位中的法决按 CD 自动轮播。

## 6. 已学法决 (xt_player_skill)

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGSERIAL | PK |
| `user_id` | BIGINT | FK → xt_user(id) |
| `skill_id` | BIGINT | FK → xt_skill(id) |
| `equipped` | BOOLEAN | 是否放入战斗槽位 |
| `slot_index` | INT | 槽位序号（1-5） |
| `source` | VARCHAR(32) | 习得来源：JADE(玉简) / REWARD(奖励) / INITIAL(初始) |
| `learned_at` | TIMESTAMP | 习得时间 |

UNIQUE(user_id, skill_id)：防重复学习。

## 7. 战斗自动施放

### 7.1 核心逻辑

```
在每场自动战斗（懒结算的一帧）中：
  for each 已放入槽位的法决：
    if skill.CD 已转好 and 法器绑定条件满足：
      触发法决效果
      重置 CD
```

### 7.2 设计原则

- **零消耗**：法决不消耗灵力/HP，纯 CD 限制
- **独立 CD**：每个法决各自独立冷却，互不干扰
- **自动轮播**：无优先级、无触发条件判断（除斩杀类），CD 好就用
- **法器约束**：绑定了法器类型的法决，若当前装备不匹配则跳过

### 7.3 首批建议法决

| 法决 | 效果 | 绑定 | CD | 说明 |
|------|------|------|-----|------|
| 御剑术 | DAMAGE | 剑 | 30s | 剑系基础法决 |
| 破空斩 | DAMAGE | 刀 | 40s | 刀系单发高伤 |
| 雷霆一击 | ARMOR_BREAK | 斧 | 60s | 斧系破甲 |
| 穿云箭 | DAMAGE | 弓 | 35s | 远程必中 |
| 伏魔棍法 | DAMAGE | 棍 | 30s | 额外克制邪祟 |
| 烈焰诀 | DAMAGE | NONE | 45s | 不绑定法器 |
| 寒冰诀 | SLOW | NONE | 60s | 不绑定法器 |
| 万剑诀 | MULTI_HIT | 剑 | 120s | 剑系高阶群伤 |

## 8. 法决玉简 (SKILL_JADE)

法决玉简是一种**可堆叠物品**，通过 `ItemType.SKILL_JADE` 标识。

`ItemTemplate.properties` 中记录关联的法决 ID：

```json
{"skill_id": 1}
```

### 使用流程

玩家在背包中使用法决玉简 → 消耗物品 → 向 `xt_player_skill` 插入记录 → 习得对应法决。

### 获取方式

- 怪物掉落（在怪物模板的 `drop_table.items` 中配置为普通物品掉落）
- 任务奖励
- 坊市购买
