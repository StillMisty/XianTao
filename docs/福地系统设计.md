## 2. 灵气系统

### 2.1 消耗公式

```
每小时消耗 = 10 + 地块数量 × 2 + Σ(灵兽等阶 × 5 × 品质消耗倍率) + Σ(阵眼等级 × 3)
```

### 2.2 上限公式

```
灵气上限 = 1000 + 聚灵核心等级 × 200
```

### 2.3 献祭收益

```
献祭收益 = 物品基础价值 × 品质乘数 × (1 + 物品等级/10)
```

| 品质 | 乘数 |
|---|---|
| 白色 | 0.5x |
| 绿色 | 1.0x |
| 蓝色 | 2.0x |
| 紫色 | 5.0x |
| 橙色 | 10.0x |

### 2.4 灵气耗尽风险

```
灵气 > 10% → 灵兽正常运作
灵气 ≤ 10% → 灵兽产出减半
灵气 = 0 → 灵兽蛰伏，开始24h倒计时
24h后仍未补充 → 灵兽逃逸（全部投入损失）
```

## 3. 网格布局 (grid_layout JSONB)

### 3.1 地块类型 (CellType)

| 类型 | 作用 |
|---|---|
| FARM (灵田) | 种植灵药，五行影响生长 |
| PEN (兽栏) | 孵化/饲养灵兽，持续产出物品 |
| NODE (阵眼) | 提升灵气产出/防御 |

### 3.2 地块等级

灵田和兽栏均有独立等级 (Lv1~5)：

| 等级 | 升级消耗 | 累积 | 加速系数 |
|---|---|---|---|
| Lv1 | — | 0 | 1.00× |
| Lv2 | 200 | 200 | 1.15× |
| Lv3 | 400 | 600 | 1.30× |
| Lv4 | 800 | 1400 | 1.45× |
| Lv5 | 1600 | 3000 | 1.60× |

**等级差加速**：`加速系数 = 1 + (地块等级 - 最低需求等级) × 0.15`

### 3.3 五行系统 (WuxingType)

金 → 水 → 木 → 火 → 土 → 金 (相生)
金 → 木 → 土 → 水 → 火 → 金 (相克)

相邻地块五行影响：

- 相生: ×1.3
- 相同: ×1.0
- 相克: ×0.7
- 跨类型加成：FARM-PEN 五行相生: ×1.2
- NODE 同元素: ×1.1

## 4. 种植系统 (灵田)

**定位**：低门槛、快周转、活跃型。

### 4.1 核心公式

```
种植消耗 = 种子 + tier × 5 灵气
生长时间 = 基础生长时间 / (五行修正 × 等级加速)
产量 = (1~3) + 核心等级/5 × 跨类型五行加成
收获消耗 = 精力 5 点
```

### 4.2 多季作物

高 tier 种子产出 `1 + tier/3` 次，地块不消失，重置生长进度。

### 4.3 枯萎机制

普通作物：成熟后超 `2×基础生长时间` 未收 → 枯萎消失。
**多季作物免除枯萎检查。**

## 5. 养殖系统 (兽栏)

**定位**：高投入、持续产出物品、AFK友好。**无喂食机制，纯灵气维持。**

### 5.1 灵兽品质 (BeastQuality)

| 品质 | 产出倍率 | 灵气消耗倍率 | 寿命倍率 | 孵化权重 |
|---|---|---|---|---|
| 凡品 | 1.0× | 1.0× | 1.0× | 900 |
| 灵品 | 1.2× | 0.85× | 1.1× | 80 |
| 仙品 | 1.5× | 0.70× | 1.2× | 18 |
| 圣品 | 2.0× | 0.55× | 1.3× | 2 |
| 神品 | 3.0× | 0.40× | 1.5× | 0 |

### 5.2 核心公式

```
孵化消耗 = tier × 200 + 200 灵气
孵化时间 = (24 + tier × 8)h / (等级加速 × 五行修正)
产出间隔 = 4h / (等级加速 × 五行修正 × 跨类型加成)
每轮产出 = tier × 品质产出倍率 × wuxingMod × auraFactor × (1 + rand(0,tier)) / 2 件
寿命 = tier × 7天 × 品质寿命倍率
善终精华 = tier × 200 × 品质产出倍率 灵气等价物
```

### 5.3 进化与品质突破

**进化（升阶）**：
- 消耗：1进化石 + `(tier+1) × 200` 灵气
- 成功率：85% + 好感加成(最高+15%)
- 10%概率连带品质升级
- 失败：灵气+进化石不退，寿命减半

**品质突破（升品）**：
- 消耗：2进化石 + `品级 × 300` 灵气
- 成功率：60% + 好感加成(最高+20%)
- 失败：灵气+进化石不退，寿命减半

### 5.4 变异系统 (MutationTrait)

| 触发 | 概率 |
|---|---|
| 孵化 | 5% |
| 进化 | 10% |
| 品质突破 | 15% |

特性池：高产(+30%)、长寿(+50%)、节能(-20%)、稀产(5%特性)、护主(+50%)、灵悟(+10%)

## 6. 天劫系统

### 6.1 周期

每 7 天触发一次。

### 6.2 攻防公式

```
天劫攻击力 = 玩家等级 × 100 + 福地规模系数 × 200
福地总防 = Σ(阵眼耐久) + Σ(灵兽战力) + 玩家STR × 10 + 福地等级 × 50
```

---

## 7. 蛰伏模式

触发：离线 ≥ 24小时 且 灵气 < 上限×10%
效果：所有产出停止，消耗降至 1 灵气/小时
