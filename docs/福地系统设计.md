## 2. 灵气系统

### 2.1 消耗公式

```
每小时消耗 = 10 + 地块数量 × 2 + 核心等级 × 5
```

### 2.2 上限公式

```
灵气上限 = 1000 + 聚灵核心等级 × 200
```

### 2.3 献祭收益

```
献祭收益 = 物品基础价值 × 品质乘数 × (1 + 物品等级/10)
```

| 品质 | 乘数    |
|----|-------|
| 白色 | 0.5x  |
| 绿色 | 1.0x  |
| 蓝色 | 2.0x  |
| 紫色 | 5.0x  |
| 橙色 | 10.0x |

## 3. 网格布局 (grid_layout JSONB)

### 3.1 坐标系统

- 初始 3x3 (9格)，坐标 `"0,0"` 到 `"2,2"`
- 聚灵核心独立于网格之外

### 3.2 地块类型 (CellType)

| 类型        | 作用          |
|-----------|-------------|
| FARM (灵田) | 种植灵药，五行影响生长 |
| PEN (兽栏)  | 饲养灵兽，饥饿值系统  |
| NODE (阵眼) | 提升灵气产出/防御   |

### 3.3 五行系统 (WuxingType)

金 → 水 → 木 → 火 → 土 → 金 (相生)
金 → 木 → 土 → 水 → 火 → 金 (相克)

相邻地块五行影响：

- 相生: 1.2x 加速
- 相同: 1.0x
- 相克: 0.8x 减速

```
生长速度修正 = Π(所有相邻地块修正系数)
```

### 3.4 产量公式

```
最终产量 = (1-3随机) + (核心等级 / 5)
```

## 4. 种植与收获

### 4.1 生长进度 (懒加载)

```
if (当前时间 ≥ 成熟时间) 进度 = 1.0
else 进度 = min(1.0, 已过秒数 / 总生长秒数)
```

实际生长时间 = 基础生长时间 / 生长速度修正系数

### 4.2 收获约束

- 进度 ≥ 1.0 才可收获
- 消耗地灵精力 5 点/块
- 收获后地块清空

---

## 5. 兽栏喂养

- 初始饥饿值 100，低于 30 停止产出
- 喂食恢复 30 点，消耗地灵精力 3 点

---

## 6. 天劫系统

### 6.1 周期

每 7 天触发一次：`下次天劫 = last_tribulation_time + 7天`

### 6.2 攻防公式

```
天劫攻击力 = 玩家等级 × 100 + 福地规模系数 × 200
福地总防   = Σ(阵眼耐久) + Σ(灵兽战力) + 玩家STR×10 + 地灵等级×50
```

| 网格  | 规模系数 |
|-----|------|
| 3x3 | 1    |
| 4x4 | 1.5  |
| 5x5 | 2    |

### 6.3 结果

**成功**: 地灵等级提升, 连胜+1, 情绪→Celebrating
**失败**: 随机地块→焦土, 连胜归零, 情绪→Angry

---

## 7. 蛰伏模式

触发：离线 ≥ 24小时 且 灵气 < 上限×10%
效果：所有产出停止，消耗降至 1 灵气/小时
恢复：上线自动解除

---

## 8. 地灵系统

### 8.1 情绪状态机 (EmotionState)

```
if (灵气 > 80% && 地块全满)       → JOYFUL
else if (精力 < 30)               → TIRED
else if (灵气 < 20% || 有焦土)     → ANGRY
else if (刚渡劫成功)               → CELEBRATING
else                              → CALM
```

### 8.2 形态阶段 (SpiritStage)

| 阶段      | 等级    | 能力         |
|---------|-------|------------|
| STAGE_1 | 1-10  | 基础对话，无福地感知 |
| STAGE_2 | 11-30 | 感知状态，提供建议  |
| STAGE_3 | 31+   | 完全人格化，主动管理 |

### 8.3 精力系统

- 上限 100，每 24 小时自动回满
- 收获: 5点/块, 喂养: 3点/次, 自动灵气: 10点/次
- 精力 < 30 → 疲劳状态，停止自动行为

---

## 9. 好友注灵

消耗自身灵气为好友福地注入灵气，每次最多 50 点。
