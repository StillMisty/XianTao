# 命令参考手册

## 1. 注册与状态

| 命令 | 说明 |
|------|------|
| `我要修仙 [道号]` | 注册新角色 |

## 9. 地灵对话

| 命令 | 说明 |
|------|------|
| `地灵 [内容]` | 与地灵自然语言交互 |
| `地灵送礼 [物品名]` | 给地灵送礼 |

**对话示例**：
- 种植：`地灵 在1号种灵芝`
- 收获：`地灵 收获 1` / `地灵 全部收获`
- 建造：`地灵 在2号建个灵田`
- 查询：`地灵 福地状态` / `地灵 灵气还有多少`
- 聊天：`地灵 你好呀`

## 10. 商铺与交易

| 命令 | 说明 |
|------|------|
| `掌柜 [内容]` | 与商铺掌柜自然语言交互（买/卖/闲聊） |
| `回收 [物品名]` | 快速出售物品给掌柜（绕过 LLM） |

**掌柜对话示例（购买）**：
- `掌柜 有什么好货？`
- `掌柜 聚气丹怎么卖？`
- `掌柜 来两瓶聚气丹`

**掌柜对话示例（出售）**：
- `掌柜 这柄剑你收不收？`
- `掌柜 我想卖聚气丹`
- `掌柜 你看这药材值多少钱？`

## 11. 秘境

| 命令 | 说明 |
|------|------|
| `秘境列表` | 查看可进入的秘境 |
| `秘境 [秘境名]` | 查看详情 / 单人进入 |
| `秘境探索 [建筑名]` | 探索当前区域的指定建筑 |
| `秘境继续` | 推进到下一区域 |
| `秘境邀请 [道号]` | 邀请组队 |
| `秘境开始` | 队长开始挑战 |
| `秘境接受 / 秘境拒绝` | 接受/拒绝组队邀请 |
| `秘境撤退` | 退出秘境并结算 |
| `秘境进度` | 当前秘境状态（区域、已探索建筑、奖励次数） |
| `秘境首通` | 全服首通记录 |

**探索流程示例**：
```
秘境 紫府秘境       → 进入外围，看到建筑列表
秘境探索 散修遗居   → 探索此建筑
...探索完所有外围建筑...
秘境继续           → 进入内围
秘境继续           → 进入核心
秘境撤退           → 结算退出
```
