## 地灵对话

| 命令 | 说明 |
|------|------|
| `地灵 [内容]` | 与地灵自然语言交互 |
| `地灵送礼 [物品名]` | 给地灵送礼 |

**对话示例**：
- 种植：`地灵 在1号种灵芝`
- 收获：`地灵 收获 1` / `地灵 全部收获`
- 建造：`地灵 在2号建个灵田`
- 查询：`地灵 福地状态` / `地灵 灵气还有多少`
- 聊天：`地灵 你好呀`

## 秘境

| 命令 | 说明 |
|------|------|
| `秘境` | 查看秘境列表 + 当前进度 |
| `秘境 [秘境名]` | 查看详情（含首通记录）/ 进入秘境 |
| `秘境探索` | 自动探索当前区域下一个未探索建筑 |
| `秘境继续` | 推进到下一区域 |
| `秘境撤退` | 退出秘境并结算 |

**探索流程示例**：
```
秘境              → 查看秘境列表
秘境 紫府秘境     → 进入外围，看到建筑列表
秘境探索          → 探索下一个建筑（战斗自动结算）
秘境探索          → 继续探索...
秘境深入          → 提示"通道已开启，输入「秘境继续」"
秘境继续          → 进入内围
秘境撤退          → 结算退出
```

组队进入秘境需先通过**组队系统**（见下节）建立队伍。
