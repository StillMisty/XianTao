# 命令参考手册

## 1. 注册与状态

| 命令 | 说明 |
|------|------|
| `我要修仙 [道号]` | 注册新角色 |
| `状态` | 查看角色完整状态 |

---

## 2. 修炼

| 命令 | 说明 |
|------|------|
| `突破` | 尝试突破（含护道加成） |
| `护道 [道号]` | 建立护道关系 |
| `护道解除 [道号]` | 解除护道关系 |
| `护道查询` | 查询护道信息 |

---

## 3. 物品与装备

| 命令 | 说明 |
|------|------|
| `背包` | 查看完整背包 |
| `背包 种子` | 查看种子 |
| `背包 装备` | 查看装备 |
| `背包 兽卵` | 查看兽卵 |
| `装备 [物品名]` | 穿戴装备 |
| `卸下 [部位]` | 卸下装备 |

---

## 4. 法决系统

| 命令 | 说明 |
|------|------|
| `法决` | 查看当前装载的法决 |
| `法决列表` | 查看已学习的全部法决 |
| `法决装载 [法决名]` | 装载法决到槽位 |
| `法决卸下 [法决名]` | 卸下法决 |
| `学习法决 [玉简名]` | 通过玉简学习法决 |

---

## 5. 地图与历练

| 命令 | 说明 |
|------|------|
| `地图` | 查看地图列表 |
| `前往 [地图名]` | 前往目标地图 |
| `历练` | 开始挂机历练 |
| `历练结算` | 结束历练并结算收益 |

---

## 6. 悬赏系统

| 命令 | 说明 |
|------|------|
| `悬赏列表` | 查看可接取的悬赏 |
| `悬赏接取 [悬赏ID]` | 接取指定悬赏 |
| `悬赏提交` | 提交完成悬赏 |
| `悬赏放弃` | 放弃当前悬赏 |

---

## 7. 福地系统

| 命令 | 说明 |
|------|------|
| `福地` | 查看福地总览状态 |
| `福地地块` | 查看地块布局详情 |
| `福地种植 [编号] [作物]` | 种植灵药 |
| `福地收取 [编号]` | 收取指定位置产出 |
| `福地建造 [编号] [类型]` | 建造地块（灵田/兽栏） |
| `福地拆除 [编号]` | 拆除地块 |
| `福地升级 [编号]` | 升级地块 |

---

## 8. 灵兽系统

| 命令 | 说明 |
|------|------|
| `福地孵化 [编号] [兽卵名]` | 在兽栏孵化兽卵 |
| `福地放生 [编号]` | 放生灵兽 |
| `福地进化 [编号] [模式]` | 进化灵兽 |
| `灵兽出战 [编号]` | 让灵兽出战 |
| `灵兽召回 [编号]` | 召回灵兽 |
| `灵兽恢复 [编号]` | 恢复灵兽状态 |

---

## 9. 地灵对话

| 命令 | 说明 |
|------|------|
| `地灵 [内容]` | 与地灵自然语言交互 |
| `地灵送礼 [物品名]` | 给地灵送礼 |

**对话示例**：
- 种植：`地灵 在1号种灵芝`
- 收获：`地灵 收获 1` / `地灵 全部收获`
- 建造：`地灵 在2号建个灵田`
- 查询：`地灵 福地状态` / `地灵 灵气还有多少`
- 聊天：`地灵 你好呀`
