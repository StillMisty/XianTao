# 命令参考手册

## 1. 注册与状态

| 命令          | 说明       | 需要注册 |
|-------------|----------|------|
| `我要修仙 [道号]` | 注册新角色    | ❌    |
| `状态`        | 查看角色完整状态 | ✅    |

---

## 2. 修炼

| 命令             | 说明                      |
|----------------|-------------------------|
| `加点 [属性] [点数]` | 分配属性点 (STR/CON/AGI/WIS) |
| `洗点`           | 重置所有属性 (72h 冷却)         |
| `突破`           | 尝试突破 (含护道加成)            |
| `护道 [道号]`      | 建立护道关系                  |
| `护道解除 [道号]`    | 解除护道关系                  |
| `护道查询`         | 查询护道信息                  |

---

## 3. 物品与装备

| 命令         | 说明     |
|------------|--------|
| `背包`       | 查看背包摘要 |
| `装备 [物品名]` | 穿戴装备   |
| `卸下 [部位]`  | 卸下装备   |

---

## 4. 地图与移动

| 命令        | 说明          |
|-----------|-------------|
| `地图`      | 查看当前地图及相邻地图 |
| `前往 [地图]` | 前往目标地图      |

## 5. 历练

| 命令     | 说明        |

## 6. 福地

| 命令             | 说明              |
|----------------|-----------------|
| `福地` / `福地状态`  | 查看福地整体状态        |
| `福地网格`         | 查看网格布局详情        |
| `福地灵气`         | 查看灵气详情          |
| `福地地灵`         | 查看地灵详情          |
| `种植 [坐标] [作物]` | 种植灵药            |
| `收获 [坐标/all]`  | 收获作物            |
| `建造 [坐标] [类型]` | 建造地块 (灵田/兽栏/阵眼) |
| `拆除 [坐标]`      | 拆除地块            |
| `献祭 [物品名/all]` | 献祭装备获取灵气        |
| `喂养 [坐标] [饲料]` | 喂养灵兽            |

**未实现**：

- `福地升级` — 升级聚灵核心
- `福地扩建` — 扩建网格
- `福地自动 [开/关]` — 自动模式切换
- `福地注灵 [道号] [数量]` — 好友注灵
- `福地访问 [道号]` — 访问好友福地
- `唤醒福地` — 解除蛰伏模式

---

## 7. 地灵对话

| 命令                     | 说明        |
|------------------------|-----------|
| `@地灵 [内容]` 或 `地灵 [内容]` | 与地灵自然语言交互 |

支持指令词（功能等价于上述福地指令）：

- 种植：`"地灵 在左上角种灵芝"` / `"地灵 种点人参"`
- 收获：`"地灵 收获 1,1"` / `"地灵 全部收获"`
- 建造：`"地灵 在2,0建个灵田"` / `"地灵 放个阵眼"`
- 献祭：`"地灵 献祭铁剑"` / `"地灵 把所有白装转化了"`
- 喂养：`"地灵 喂一下2,0的灵兽"`
- 查询：`"地灵 福地状态"` / `"地灵 灵气还有多少"`
- 聊天：`"地灵 你好呀"` / `"地灵 今天辛苦了"`
