# 历练系统设计

## 1. 概述

历练系统是玩家挂机获取经验和物品的核心玩法。玩家在历练区地图中自动遇怪战斗，结算时获得经验、物品和铜币奖励。

**核心流程：**
1. 玩家在历练区地图使用 `历练` 命令开始历练
2. 系统记录开始时间，玩家状态变为 `EXERCISING`
3. 玩家使用 `历练结算` 命令结束历练
4. 系统计算历练时长，模拟战斗，结算奖励

**设计原则：**
- 历练只能在用户当前所在地图进行，不支持指定其他地图
- 旅行系统提供移动成本，历练系统提供挂机收益，两者形成完整的游戏循环

## 3. 核心组件

### 3.1 服务层

**TrainingService** - 历练核心服务
- `startTraining(PlatformType, String openId)` - 公开API，开始历练
- `endTraining(PlatformType, String openId)` - 公开API，结束历练
- `startTraining(Long userId)` - 内部API
- `endTraining(Long userId)` - 内部API
- `calculateTrainingRewards(Long userId)` - 预览收益
- `calculateEfficiencyMultiplier(int agility)` - 效率倍率计算
- `calculateLevelDecayMultiplier(int playerLevel, int mapLevel)` - 等级衰减计算

**CombatService** - 战斗服务
- `simulateTraining(Long userId, int durationMinutes)` - 历练战斗模拟
- `buildPlayerTeam(User user)` - 构建玩家队伍（包含灵兽）
- `calculateGearScore(Long userId)` - 计算装备评分

### 3.2 战斗引擎

**CombatEngine** - 战斗引擎接口
- `simulate(BattleContext context)` - 执行战斗模拟

**DefaultCombatEngine** - 默认战斗引擎实现
- 支持所有技能效果类型
- 支持Buff/Debuff系统
- 支持灵兽AI行为

**BattleContext** - 战斗上下文
- 封装战斗所需的所有状态和配置

### 3.3 遇怪计算

**EncounterCalculator** - 遇怪计算器
- `calculateInterval(mapLevel, playerLevel, gearScore)` - 计算动态遇怪间隔
- `calculateEncounterChances(durationMinutes, interval)` - 计算遇怪次数
- `calculateEncounterChance(interval)` - 计算遇怪概率

**HighlightBattleDetector** - 高光战斗检测器
- `detectHighlight(battleResult, battleIndex)` - 检测是否为高光战斗

### 3.4 灵兽AI

**BeastAI** - 灵兽AI接口
- `decideAction(beast, allies, enemies)` - 决定行动
- `selectTarget(beast, enemies, allies)` - 选择目标
- `selectSkill(beast, target, allies, enemies)` - 选择技能

**DefaultBeastAI** - 默认灵兽AI实现
- 配合玩家策略
- 优先攻击威胁最大的敌人
- 智能使用技能

### 3.5 Buff/Debuff系统

**Buff** - Buff实体
- 支持12种效果类型
- 支持叠加和刷新

**BuffManager** - Buff管理器
- 管理战斗单位身上的Buff/Debuff
- 计算属性修正

## 4. 数据结构

### 4.1 TrainingRewardVO - 历练奖励

```java
TrainingRewardVO
├── userId                    // 用户ID
├── mapId                     // 历练地图ID
├── mapName                   // 历练地图名称
├── durationMinutes           // 历练时长（分钟）
├── efficiencyMultiplier      // 效率倍率（基于敏捷）
├── levelDecayMultiplier      // 等级衰减倍率
├── exp                       // 获得的经验
├── items                     // 获得的物品
└── summary                   // 总奖励描述
```

### 4.2 BattleResultVO - 战斗结果

```java
BattleResultVO
├── winner                    // 获胜方
├── rounds                    // 战斗回合数
├── playerHpChange            // 玩家HP变化
├── beastHpChanges            // 灵兽HP变化
├── monsterHpChanges          // 怪物HP变化
├── damageDealt               // 伤害统计
├── skillProcs                // 技能触发记录
├── combatLog                 // 战斗日志
├── drops                     // 掉落物品
├── expGained                 // 获得的经验
└── summary                   // 战斗摘要
```

### 4.3 TrainingStatistics - 历练统计

```java
TrainingStatistics
├── totalEncounters           // 遇敌场次
├── totalKills                // 击杀怪物数量

├── totalRounds               // 总战斗回合数
├── beastStats                // 灵兽战斗统计
├── skillProcStats            // 法决触发统计
└── highlightBattleIndex      // 高光战斗索引
```

## 5. 计算公式

### 5.1 动态遇怪间隔

```
encounterInterval = baseInterval × (1 - difficultyFactor) × (1 - levelFactor) × (1 - gearFactor)
```

- `baseInterval` = 10分钟
- `difficultyFactor` = min(0.5, mapLevel × 0.005)
- `levelFactor` = min(0.3, playerLevel × 0.0015)
- `gearFactor` = min(0.2, gearScore × 0.0002)
- 最小间隔：2分钟

### 5.2 等级衰减

```
expMultiplier = max(0.1, 1 - max(0, (playerLevel - mapLevel - 10)) × 0.05)
```

- 玩家等级 ≤ 地图等级 + 10：无衰减（100%经验）
- 玩家等级 > 地图等级 + 10：每高1级衰减5%
- 最低衰减到10%

### 5.3 效率倍率

```
efficiencyMultiplier = 1.0 + (agility × 0.01)
```

### 5.4 装备评分

```
gearScore = sum(equipment.finalAttack + equipment.finalDefense × 2) + rarityBonus
```

稀有度加成：
- 普通 (common) → +10
- 稀有 (rare) → +40
- 史诗 (epic) → +80
- 传说 (legendary) → +160

## 6. 输出格式

### 6.1 普通战斗统计

```
【历练结算】幽暗沼泽 | 30分钟 | 效率1.2x
遇敌8场 | 击杀22只 | 战败1场
经验+460 | 灵草×5 兽骨×2 破损的木剑×1
灵兽：火羽鸡 +120exp HP42→15 | 灵狐 +60exp HP100→85
```

### 6.2 高光战斗（LLM渲染）

```
【历练结算】幽暗沼泽 | 30分钟 | 效率1.2x
遇敌8场 | 击杀22只 | 战败1场
经验+460 | 灵草×5 兽骨×2 破损的木剑×1
灵兽：火羽鸡 +120exp HP42→15 | 灵狐 +60exp HP100→85

━━━ 高光时刻 ━━━
第5场战斗，你遭遇了一只剧毒蟾蜍王。它喷出的毒雾几乎遮蔽了你的视线，
但火羽鸡及时喷出烈焰，将毒雾驱散。你抓住机会，御剑术触发，
一剑刺穿了它的要害。这场战斗持续了5个回合，最终你以微弱优势胜出。
```

### 6.3 高光战斗触发条件

- 战斗回合数超过10回合
- 双方血量都低于30%（势均力敌）
- 触发了稀有技能或连击（>=3次）
