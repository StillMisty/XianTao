# 物品与装备系统

## 1. 物品模板 (xt_item_template)

### 1.1 表结构

| 字段                | 类型           | 说明         |
|-------------------|--------------|------------|
| `id`              | BIGSERIAL    | PK         |
| `name`            | VARCHAR(128) | 物品名称       |
| `type`            | VARCHAR(32)  | 物品类型       |
| `rarity`          | VARCHAR(32)  | 稀有度        |
| `slot`            | VARCHAR(32)  | 装备部位       |
| `equip_level`     | INT          | 装备等级       |
| `base_stat_bonus` | JSONB        | 基础属性加成     |
| `base_attack`     | INT          | 基础攻击       |
| `base_defense`    | INT          | 基础防御       |
| `drop_weight`     | JSONB        | 掉落权重       |
| `tags`            | JSONB        | 物品标签 (GIN) |
| `grow_time`       | INT          | 生长时间(小时)   |
| `yield_id`        | VARCHAR(64)  | 产出模板ID     |
| `survive_rate`    | INT          | 存活率(%)     |
| `max_stack`       | INT          | 最大堆叠       |
| `description`     | TEXT         | 描述         |

### 1.2 JSONB 示例

**base_stat_bonus**: `{"str": 5, "con": 3}`
**drop_weight**: `{"BROKEN": 50, "COMMON": 30, "RARE": 15, "EPIC": 4, "LEGENDARY": 1}`
**tags**: `["ore", "metal", "forge_base"]`

## 2. 装备实例 (xt_equipment)

| 字段                   | 类型           | 说明         |
|----------------------|--------------|------------|
| `id`                 | BIGSERIAL    | PK         |
| `user_id`            | BIGINT       | 持有者        |
| `template_id`        | BIGINT       | 模板ID       |
| `name`               | VARCHAR(128) | 实际名称       |
| `slot`               | VARCHAR(32)  | 部位         |
| `rarity`             | VARCHAR(32)  | 稀有度        |
| `stat_bonus`         | JSONB        | 属性加成 (GIN) |
| `attack_bonus`       | INT          | 攻击加成       |
| `defense_bonus`      | INT          | 防御加成       |
| `equipped`           | BOOLEAN      | 已装备        |
| `quality_multiplier` | DOUBLE       | 品质乘数       |
| `affixes`            | JSONB        | 随机词条 (GIN) |
| `forge_level`        | INT          | 锻造等级 (默认0) |

### 2.1 稀有度体系

| 稀有度       | 品质乘数 | 颜色 |
|-----------|------|----|
| BROKEN    | 0.5x | 灰色 |
| COMMON    | 1.0x | 白色 |
| RARE      | 1.5x | 绿色 |
| EPIC      | 2.5x | 蓝色 |
| LEGENDARY | 4.0x | 橙色 |

### 2.2 属性计算

```
最终攻击 = 模板.base_attack × quality_multiplier
最终防御 = 模板.base_defense × quality_multiplier
最终属性 = 模板.base_stat_bonus × quality_multiplier
```

### 2.3 装备部位 (EquipmentSlot)

WEAPON, ARMOR, ACCESSORY — 共 3 个部位。

### 2.4 词条 (AffixType)

| 词条            | 效果         |
|---------------|------------|
| STRENGTH      | 力量 +X      |
| CONSTITUTION  | 体质 +X      |
| AGILITY       | 敏捷 +X      |
| WISDOM        | 智慧 +X      |
| LIFE_STEAL    | 伤害的5%转化为HP |
| TREASURE_HUNT | 挂机极品掉率+5%  |

## 3. 堆叠物品 (xt_inventory_item)

| 字段            | 类型        | 说明       |
|---------------|-----------|----------|
| `id`          | BIGSERIAL | PK       |
| `user_id`     | BIGINT    | 持有者      |
| `template_id` | BIGINT    | 模板ID     |
| `quantity`    | INT       | 数量       |
| `tags`        | JSONB     | 标签 (GIN) |

UNIQUE(user_id, template_id)：同模板物品合并存储。

## 4. 物品类型 (ItemType)

| 类型         | 说明  | 前道        |
|------------|-----|-----------|
| EQUIPMENT  | 装备  | 穿戴提供属性    |
| MATERIAL   | 材料  | 锻造、布阵     |
| SEED       | 种子  | 福地种植      |
| SPIRIT_EGG | 灵蛋  | 福地孵化      |
| CONSUMABLE | 消耗品 | 使用恢复      |
| HERB       | 草药  | 炼药材料      |
| POTION     | 丹药  | HP恢复、Buff |
| GIFT       | 礼物  | NPC送礼     |

---

## 5. 背包管理

- 初始容量：50 格（硬编码，暂未可配置）
- 装备：每件独立实例，不堆叠
- 材料/消耗品：按模板合并，受 `max_stack` 限制
- 添加物品时自动堆叠，背包满时返回失败

---

## 6. 装备穿戴流程

1. 验证物品存在、类型为装备、等级要求满足
2. 检查部位是否已有装备 — 有则先卸下
3. 标记新装备 `equipped=true`
4. 计算属性变化，更新用户实体
5. 返回 `AttributeChange`（各属性变化明细）
