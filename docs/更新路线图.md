#### 掉落表格式

```json
{
  "equipment": {"1": 50, "2": 30},
  "items": {"10": 80, "20": 30}
}
```

- `equipment`: key = EquipmentTemplate ID, value = 权重，击杀时 Roll 品质生成 Equipment
- `items`: key = ItemTemplate ID, value = 权重（含材料/丹药/法决玉简），击杀时发放堆叠物品

#### 地图遇怪池

`MapNode.monster_encounters JSONB`：`{"template_id": encounter_weight}`

`MapNode.encounter_size JSONB`：`{"min": 1, "max": 3}`

遇敌时，先按 `monster_encounters` 加权选模板，再按 `encounter_size` 随机生成 `[min, max]` 只怪物组成一波投入战斗。

#### 拆除兽栏的行为

- 拆除兽栏时 **不删除** `xt_beast` 记录，仅将 `penned_cell_id` 置为 null
- 栏外灵兽：不可生产（失去产出功能），但 **可出战战斗**
- 新建兽栏后可重新将栏外灵兽放入（`penned_cell_id = 新建栏位编号`）
- 栏外灵兽上限 = 玩家等级 × 1（防止无限囤积无栏灵兽）
- 彻底放生走 `#福地放生` 指令，与拆除兽栏无关

#### 属性公式

```
品质倍率: 凡品 ×0.8 / 灵品 ×1.0 / 仙品 ×1.3 / 圣品 ×1.6 / 神品 ×2.0

Lv.1 基础: atk=10, def=8, hp=100
每级成长: atk+=3q, def+=2q, hp+=15q
```

#### 阵亡休养

```
凡品 30min / 灵品 1h / 仙品 2h / 圣品 4h / 神品 8h
```

休养期间不可出战。
