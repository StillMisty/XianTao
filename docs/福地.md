## 2. 灵气系统

### 2.1 消耗公式

```
每小时消耗 = 已占地块数 × 2 + Σ(灵兽等阶 × 5 × 品质消耗倍率) + Σ(阵眼等级 × 3)
```

### 2.2 上限公式

```
灵气上限 = 1000 + 天劫胜利积累
```

- 每次天劫胜利：`灵气上限 += 连胜次数 × 100`

### 2.3 献祭收益

```
献祭收益 = 物品基础价值 × 品质乘数 × (1 + 物品等级/20)

物品基础价值 = 装备.最终攻击 + 装备.最终防御
```

| 品质 | 乘数 |
|---|---|
| 白色 | 0.5x |
| 绿色 | 1.0x |
| 蓝色 | 2.0x |
| 紫色 | 5.0x |
| 橙色 | 10.0x |

### 2.4 灵气枯竭机制

```
灵气 = 0 → 除献祭外所有功能不可用
灵兽连续断灵 24h → 逃逸（全部投入损失）
灵气恢复后 → 功能自动恢复，灵兽逃逸倒计时重置
```

## 3. 地块布局 (grid_layout JSONB)

### 3.1 数据结构

```json
{
  "cells": [
    { "cell_id": 1, "type": "empty" },
    { "cell_id": 2, "type": "farm", "cell_level": 1, "crop_id": 101, ... },
    { "cell_id": 3, "type": "pen", "cell_level": 1, "beast_name": "火凤", ... },
    { "cell_id": 4, "type": "node", "cell_level": 1, "level": 1, "durability": 100 },
    { "cell_id": 5, "type": "empty" }
  ]
}
```

- `cell_id` 从 1 开始连续递增，唯一标识地块
- `type: "empty"` 的空地块只有 `cell_id` 和 `type` 两个字段
- 建造/拆除操作不改变 cell_id，empty 保留槽位

### 3.2 地块类型 (CellType)

| 类型 | 作用 |
|---|---|
| FARM (灵田) | 种植灵药 |
| PEN (兽栏) | 孵化/饲养灵兽，持续产出物品 |
| NODE (阵眼) | 提升防御力 |

### 3.3 地块等级

灵田和兽栏均有独立等级 (Lv1~5)：

| 等级 | 升级消耗 | 累积 | 加速系数 |
|---|---|---|---|
| Lv1 | — | 0 | 1.00× |
| Lv2 | 200 | 200 | 1.15× |
| Lv3 | 400 | 600 | 1.30× |
| Lv4 | 800 | 1400 | 1.45× |
| Lv5 | 1600 | 3000 | 1.60× |

**等级差加速**：`加速系数 = 1 + (地块等级 - 最低需求等级) × 0.15`

### 3.4 地块扩建

- 初始地块数：**3**（cell_id = 1, 2, 3）
- 增长公式：`maxCells = 3 + floor(tribulationStage / 3)`
- 扩建方式：**自动解锁**，无消耗
- 触发时机：任何操作调用 `getFudiByUserId()` 时懒加载

## 4. 种植系统 (灵田)

**定位**：低门槛、快周转、活跃型。

### 4.1 核心公式

```
种植消耗 = 种子 × 1 + tier × 5 灵气
生长时间 = 基础生长时间 / 等级加速系数
产量 = (1~3) + 劫数/5
收获消耗 = 精力 5 点
```

### 4.2 多季作物

高 tier 种子产出 `1 + tier/3` 次，地块不消失，重置生长进度。

### 4.3 枯萎机制

普通作物：成熟后超 `2×基础生长时间` 未收 → 枯萎消失。
**多季作物免除枯萎检查。**

## 5. 养殖系统 (兽栏)

**定位**：高投入、持续产出物品、AFK友好。**无喂食机制，纯灵气维持。**

### 5.1 灵兽品质 (BeastQuality)

| 品质 | 产出倍率 | 灵气消耗倍率 | 寿命倍率 | 孵化权重 |
|---|---|---|---|---|
| 凡品 | 1.0× | 1.0× | 1.0× | 900 |
| 灵品 | 1.2× | 0.85× | 1.1× | 80 |
| 仙品 | 1.5× | 0.70× | 1.2× | 18 |
| 圣品 | 2.0× | 0.55× | 1.3× | 2 |
| 神品 | 3.0× | 0.40× | 1.5× | 0 |

### 5.2 核心公式

```
孵化消耗 = tier × 200 + 200 灵气
孵化时间 = (24 + tier × 8)h / 等级加速系数
产出间隔 = 4h / 等级加速系数
每轮产出 = tier × 品质产出倍率 × auraFactor × (1 + rand(0,tier)) / 2 件
寿命 = tier × 7天 × 品质寿命倍率
善终精华 = tier × 200 × 品质产出倍率 灵气等价物
```

### 5.3 进化与品质突破

**进化（升阶）**：
- 消耗：1 进化石 + `(tier+1) × 200` 灵气
- 成功率：85% + 好感加成（最高+15%）
- 10%概率连带品质升级
- 失败：灵气+进化石不退，寿命减半

**品质突破（升品）**：
- 消耗：2 进化石 + `品级 × 300` 灵气
- 成功率：60% + 好感加成（最高+20%）
- 失败：灵气+进化石不退，寿命减半

### 5.4 变异系统 (MutationTrait)

| 触发 | 概率 |
|---|---|
| 孵化 | 5% |
| 进化 | 10% |
| 品质突破 | 15% |

特性池：高产(+30%)、长寿(+50%)、节能(-20%)、稀产(5%特性)、护主(+50%)、灵悟(+10%)

## 6. 天劫系统

### 6.1 周期

每 7 天触发一次（懒求值）。

### 6.2 攻防公式

```
天劫攻击力 = 玩家等级 × 80 + 劫数 × 200
福地总防 = Σ(阵眼耐久) + Σ(灵兽战力×护主) + 玩家STR × 10 + 劫数 × 50
```

### 6.3 天劫胜利

- 连胜 +1、好感 +5、灵气上限增长（+ 连胜 × 100）、劫数 +1
- 情绪 → EXCITED

### 6.4 天劫失败

- 连胜归零
- 按差额比例随机清空地块（重置为 empty，保留 cell_id）
- 好感降低
- 情绪 → ANGRY
