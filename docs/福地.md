## 2. 地块实体 (xt_fudi_cell)

地块存储在独立表中，每个地块一行记录。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | BIGSERIAL | PK |
| `fudi_id` | BIGINT | 所属福地 |
| `cell_id` | INT | 地块编号（从1开始） |
| `cell_type` | VARCHAR(16) | 地块类型：empty/farm/pen |
| `cell_level` | INT | 地块等级（1-5） |
| `config` | JSONB | 建筑专有属性 |
| `create_time` | TIMESTAMP | 创建时间 |
| `update_time` | TIMESTAMP | 更新时间 |

**唯一约束**：`(fudi_id, cell_id)`

## 3. 地灵实体 (xt_spirit)

地灵与福地 1:1 绑定，存储在独立的 `xt_spirit` 表中。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | BIGSERIAL | PK |
| `fudi_id` | BIGINT | 所属福地 (UNIQUE) |
| `form_id` | INT | 形态ID (FK → xt_spirit_form) |
| `mbti_type` | VARCHAR(4) | MBTI人格 |
| `energy` | INT | 地灵精力 |
| `affection` | INT | 好感度 |
| `affection_max` | INT | 好感度上限 (默认1000) |
| `emotion_state` | VARCHAR(20) | 情绪状态 |
| `last_energy_update` | TIMESTAMP | 上次精力更新时间 |
| `last_gift_time` | TIMESTAMP | 上次送礼时间 |

## 6. 地块等级

灵田和兽栏均有独立等级 (Lv1~5)：

| 等级 | 升级消耗 | 累积 | 加速系数 |
|---|---|---|---|
| Lv1 | — | 0 | 1.00× |
| Lv2 | 200 | 200 | 1.15× |
| Lv3 | 400 | 600 | 1.30× |
| Lv4 | 800 | 1400 | 1.45× |
| Lv5 | 1600 | 3000 | 1.60× |

加速系数公式：`加速系数 = 1 + (地块等级 - 最低需求等级) × 0.15`，表格为预计算结果。

---

## 7. 地块扩建

- 初始地块数：**3**（cell_id = 1, 2, 3）
- 增长公式：`maxCells = 3 + floor(tribulationStage / 3)`
- 扩建方式：**自动解锁**，无消耗
- 触发时机：任何操作调用 `getFudiByUserId()` 时懒加载

## 8. 种植系统 (灵田)

**定位**：低门槛、快周转、活跃型。

### 8.1 核心公式

```
种植消耗 = 种子 × 1 + tier × 5 灵石
生长时间 = 基础生长时间 / 等级加速系数
产量 = (1~3) + 劫数/5
收获消耗 = 地灵精力 5 点
```

### 8.2 多季作物

高 tier 种子产出 `1 + tier/3` 次，地块不消失，重置生长进度。

### 8.3 枯萎机制

普通作物：成熟后超 `2×基础生长时间` 未收 → 枯萎消失。
**多季作物免除枯萎检查。**

## 9. 天劫系统

### 9.1 触发机制

- **主动渡劫**：玩家可随时发起渡劫
- **强制渡劫**：距上次劫后 7 天自动触发（`getFudiStatus` 时检查）

**主动渡劫的优势：**
- 可以提前准备（灵兽出战、提升装备等）
- 避免在不利时机被强制渡劫

### 9.2 战斗机制

天劫化身（`TribulationBoss`）实现 `Combatant` 接口，接入通用战斗引擎进行回合制战斗。

**Boss 属性缩放** — 基于防守方队伍总战力动态缩放，天然支持未来多人组队：

```
BossHP  = 防守方总HP × (1.0 + 劫数 × 0.5)
Boss攻  = 防守方均攻 × (0.6 + 劫数 × 0.15)
Boss防  = 防守方均防 × (0.4 + 劫数 × 0.10)
Boss速  = 防守方均速 × (0.7 + 劫数 × 0.08)
```

防守方队伍 = 玩家 + 所有出战灵兽（由 `CombatService.buildPlayerTeam()` 构建）。

**怜悯模式**：地灵好感度 ≥ 800 时触发，Boss 全属性 × 0.7。

### 9.3 战斗结果

| 结果 | 条件 | 效果 |
|------|------|------|
| **胜利** | 防守方击杀 Boss | 劫数+1、连胜+1、好感+5、灵石奖励（连胜×100）、情绪→EXCITED |
| **怜悯庇护** | 防守方败 + 好感≥800 | 劫数+1、连胜+1、灵石奖励、精力归零、情绪→FATIGUED |
| **失败** | 防守方败 + 好感<800 | 劫数/连胜不变、地块摧毁、好感下降、情绪→ANGRY |

### 9.4 地块摧毁规则

摧毁数量与战斗表现挂钩（Boss 剩余 HP 越少，输得越体面）：

| Boss 剩余HP | 摧毁数量 |
|-------------|---------|
| ≥ 50% | 60% 已占用地块 |
| 20% ~ 50% | 30% 已占用地块 |
| < 20% | 1 块 |
