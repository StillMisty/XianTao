# 灵兽系统设计

## 1. 灵兽品质 (BeastQuality)

| 品质 | 产出倍率 | 孵化权重 |
|---|---|---|
| 凡品 | 1.0× | 900 |
| 灵品 | 1.2× | 80 |
| 仙品 | 1.5× | 18 |
| 圣品 | 2.0× | 2 |
| 神品 | 3.0× | 0 |

神品孵化权重为 0，仅可通过品质突破（圣品→神品）获得。

---

## 2. 核心公式

```
孵化消耗 = tier × 200 + 200 灵石
孵化时间 = (24 + tier × 8)h / 兽栏等级加速系数
产出间隔 = 4h / 兽栏等级加速系数
每轮产出 = tier × 品质产出倍率 × (1 + rand(0,tier)) / 2 件
HP最大值 = tier × 200
恢复消耗 = 缺失HP × 0.1 灵石
```

---

## 3. 进化与品质突破

### 进化（升阶）

- 消耗：1 进化石 + `(tier+1) × 200` 灵石
- 成功率：85% + 好感加成（最高+15%）
- 10%概率连带品质升级
- 失败：进化石+灵石不退

### 品质突破（升品）

- 消耗：2 进化石 + `品级 × 300` 灵石
- 成功率：60% + 好感加成（最高+20%）
- 失败：进化石+灵石不退

## 4. 变异系统 (MutationTrait)

| 触发 | 概率 |
|---|---|
| 孵化 | 5% |
| 进化 | 10% |
| 品质突破 | 15% |

特性池：高产(+30%)、节能(-20%)、稀产(5%特性)、护主(+50%)、灵悟(+10%)

## 5. 出战与恢复

灵兽在栏内持续产出。玩家可将灵兽**取出随行战斗**。

| 状态 | 产出 | 天劫防御 | HP恢复 |
|---|---|---|---|
| 栏内正常 | ✓ | ✗ | — |
| 出战 | ✗ | ✓ | — |
| 回栏休养 | ✗ | ✗ | 消耗灵石恢复 |

- **出战上限**：最多 2 只灵兽同时出战
- **出战规则**：出战灵兽占据原兽栏地块，不可用于孵化新灵兽
- **HP归零**：灵兽自动回栏，进入休养状态，不产出
- **恢复**：消耗灵石瞬间回满 HP，恢复后自动恢复产出
- **召回**：手动召回出战灵兽，回栏后立即恢复产出（若 HP 满）

出战换取天劫防御加成，但放弃该栏位产出。

---

## 6. 拆除兽栏

拆除兽栏时灵兽 **不会消失**，而是退为"栏外休憩"状态（`penned_cell_id = null`）：
- 不可生产，但 **可出战战斗**
- 新建兽栏后可重新入栏
- 栏外上限 = 玩家等级 × 1 只
- 彻底放生请使用 `#福地放生` 指令
- 栏外休憩的灵兽 **不计入** 出战上限（2 只）

## 8. 灵石消耗一览

| 操作 | 灵石消耗 |
|---|---|
| 孵化灵兽 | `tier × 200 + 200` |
| 灵兽进化 | `(tier + 1) × 200` |
| 品质突破 | `品级 × 300` |
| 灵兽恢复 HP | `缺失HP × 0.1` |
