# 灵兽系统设计

## 1. 灵兽实体 (xt_beast)

灵兽是独立的实体，存储在 `xt_beast` 表中。兽栏地块中只存储兽栏配置（等级等），灵兽通过 `penned_cell_id` 关联所在兽栏。

### 1.1 表结构

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | BIGSERIAL | PK |
| `user_id` | BIGINT | 所属用户 |
| `fudi_id` | BIGINT | 所属福地 |
| `template_id` | BIGINT | 模板ID |
| `beast_name` | VARCHAR(128) | 灵兽名称 |
| `tier` | INT | 等阶 |
| `quality` | VARCHAR(32) | 品质 |
| `is_mutant` | BOOLEAN | 是否变异 |
| `mutation_traits` | JSONB | 变异特性列表 |
| `level` | INT | 等级 |
| `exp` | INT | 经验 |
| `attack` | INT | 攻击力 |
| `defense` | INT | 防御力 |
| `max_hp` | INT | 最大HP |
| `hp_current` | INT | 当前HP |
| `skills` | JSONB | 技能列表 |
| `is_deployed` | BOOLEAN | 是否出战 |
| `recovery_until` | TIMESTAMP | 恢复截止时间（休养中） |
| `penned_cell_id` | INT | 所在兽栏地块ID（null = 栏外休憩） |
| `birth_time` | TIMESTAMP | 出生时间 |
| `evolution_count` | INT | 进化次数 |
| `create_time` / `update_time` | TIMESTAMP | 时间戳 |

### 1.2 与兽栏的关系

```
xt_fudi.grid_layout.cells[].pen  →  兽栏配置（等级、容量等）
xt_beast.penned_cell_id          →  关联兽栏地块
```

- 灵兽卵在兽栏中孵化，孵化完成后生成 `xt_beast` 记录
- 灵兽通过 `penned_cell_id` 关联到具体的兽栏地块
- `penned_cell_id = null` 表示灵兽在栏外休憩（兽栏被拆除时）

## 2. 灵兽品质 (BeastQuality)

| 品质 | 产出倍率 | 孵化权重 |
|---|---|---|
| 凡品 | 1.0× | 900 |
| 灵品 | 1.2× | 80 |
| 仙品 | 1.5× | 18 |
| 圣品 | 2.0× | 2 |
| 神品 | 3.0× | 0 |

神品孵化权重为 0，仅可通过品质突破（圣品→神品）获得。

---

## 3. 核心公式

```
孵化消耗 = tier × 200 + 200 灵石
孵化时间 = (24 + tier × 8)h / 兽栏等级加速系数
产出间隔 = 4h / 兽栏等级加速系数
每轮产出 = tier × 品质产出倍率 × (1 + rand(0,tier)) / 2 件
HP最大值 = tier × 200
恢复消耗 = 缺失HP × 0.1 灵石
```

## 4. 灵兽状态

### 4.1 状态机

```
孵化中 → 正常（在栏） → 出战 → 战斗受伤 → 回栏休养 → 正常（在栏）
                ↓
            栏外休憩（兽栏拆除）
```

### 4.2 状态说明

| 状态 | 条件 | 产出 | 天劫防御 | HP恢复 |
|---|---|---|---|---|
| 孵化中 | `is_incubating = true` | ✗ | ✗ | — |
| 正常（在栏） | HP满，非出战 | ✓ | ✗ | — |
| 出战 | `is_deployed = true` | ✗ | ✓ | — |
| 休养中 | HP < max，或 `recovery_until` 未过期 | ✗ | ✗ | 消耗灵石恢复 |
| 栏外休憩 | `penned_cell_id = null` | ✗ | ✗ | 可出战 |

### 4.3 出战规则

- **出战上限**：最多 2 只灵兽同时出战
- **出战灵兽**：占据原兽栏地块，不可用于孵化新灵兽
- **HP归零**：灵兽自动回栏，进入休养状态，不产出
- **恢复**：消耗灵石瞬间回满 HP，恢复后自动恢复产出
- **召回**：手动召回出战灵兽，回栏后立即恢复产出（若 HP 满）

### 4.4 栏外休憩

拆除兽栏时灵兽 **不会消失**，而是退为"栏外休憩"状态（`penned_cell_id = null`）：
- 不可生产，但 **可出战战斗**
- 新建兽栏后可重新入栏
- 栏外上限 = 玩家等级 × 1 只
- 栏外休憩的灵兽 **不计入** 出战上限（2 只）

## 5. 进化与品质突破

### 5.1 进化（升阶）

- 消耗：1 进化石 + `(tier+1) × 200` 灵石
- 成功率：85% + 好感加成（最高+15%）
- 10%概率连带品质升级
- 失败：进化石+灵石不退

### 5.2 品质突破（升品）

- 消耗：2 进化石 + `品级 × 300` 灵石
- 成功率：60% + 好感加成（最高+20%）
- 失败：进化石+灵石不退

## 6. 变异系统 (MutationTrait)

| 触发 | 概率 |
|---|---|
| 孵化 | 5% |
| 进化 | 10% |
| 品质突破 | 15% |

特性池：高产(+30%)、节能(-20%)、稀产(5%特性)、护主(+50%)、灵悟(+10%)

## 7. 灵石消耗一览

| 操作 | 灵石消耗 |
|---|---|
| 孵化灵兽 | `tier × 200 + 200` |
| 灵兽进化 | `(tier + 1) × 200` |
| 品质突破 | `品级 × 300` |
| 灵兽恢复 HP | `缺失HP × 0.1` |
