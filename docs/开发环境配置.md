# 开发环境配置

| 硬件组件 | 最低配置要求 | 推荐配置 | 说明 |
|---------|------------|---------|------|

## 关键配置说明

### 运行时环境变量

以下环境变量需要在运行前配置:

| 变量名 | 默认值 | 说明 |
|-------|-------|------|
| `OPENAI_API_KEY` | `sk-placeholder` | DeepSeek API密钥 |
| `OPENAI_BASE_URL` | `https://api.deepseek.com` | API基础URL |
| `OPENAI_MODEL` | `deepseek-chat` | 使用的模型名称 |

### 数据库配置

```yaml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/xiantao
    username: postgres
    password: 123456
    driver-class-name: org.postgresql.Driver
    hikari:
      minimum-idle: 10
      maximum-pool-size: 50
      idle-timeout: 30000
      max-lifetime: 1800000
      connection-timeout: 30000
      pool-name: XianTaoHikariCP
```

**连接池参数说明**:
- 最小空闲连接数: 10
- 最大连接数: 50
- 空闲超时: 30秒
- 连接最大生命周期: 30分钟
- 连接超时: 30秒

### JVM参数

运行和测试时需要添加以下JVM参数:

```bash
--enable-native-access=ALL-UNNAMED
```

该参数启用本地访问权限,为SimBot和Spring AI的底层库所必需。

### 虚拟线程配置

项目已启用Java 25虚拟线程特性:

```yaml
spring:
  threads:
    virtual:
      enabled: true
```

**注意**: 虚拟线程特性要求JDK 25+,不支持降级到早期版本。

### Maven仓库镜像

项目配置了阿里云Maven镜像以加速依赖下载:

```kotlin
repositories {
    maven { url = uri("https://maven.aliyun.com/repository/public") }
    maven { url = uri("https://maven.aliyun.com/repository/spring") }
    mavenCentral()
}
```

### Gradle配置优化

启用配置缓存以提升构建性能:

```properties
# gradle.properties
org.gradle.configuration-cache=true
```

## 注意事项

1. **PostgreSQL优化**: PostgreSQL 18需启用GIN索引以优化JSONB查询性能,所有JSONB字段都应创建对应的GIN索引。

2. **网络稳定性**: SimBot需要稳定的网络连接以保持与QQ平台的长连接,建议使用有线网络或稳定的WiFi环境。

3. **AI API配额**: AI功能依赖外部DeepSeek API,需确保:
   - 网络可达性(可能需要代理配置)
   - API配额充足
   - 密钥安全存储(不要提交到版本控制)

4. **数据库初始化**: 首次运行前需手动创建`xiantao`数据库,Flyway会自动执行迁移脚本。

5. **端口占用**: 确保以下端口未被占用:
   - PostgreSQL: 5432
   - Spring Boot Web: 8080(默认,可在配置中修改)
   - OneBotV11 WebSocket: 根据机器人配置而定

6. **开发工具推荐**:
   - 数据库管理: pgAdmin 4 或 DBeaver
   - API测试: Postman 或 Apifox
   - 图表绘制: draw.io (项目已集成相关技能)

## 快速启动检查清单

在开始开发前,请确认以下项已完成:

- [ ] 安装JDK 25并配置JAVA_HOME环境变量
- [ ] 安装PostgreSQL 18并创建`xiantao`数据库
- [ ] 配置`.env`文件或系统环境变量中的`OPENAI_API_KEY`
- [ ] 克隆项目代码并执行`./gradlew build`验证依赖下载
- [ ] 配置SimBot机器人配置文件(`src/main/resources/simbot-bots/`)
- [ ] 运行应用并检查日志无错误
